# 🤝 Contributing to Hey Dad

We're thrilled you're interested in helping make **Hey Dad** even better! Whether you're fixing a bug, adding a new feature, or improving documentation, your contributions are welcome.

## 🚀 How to Contribute

1. **Fork the Repository:**
   Go to [https://github.com/mrbrightsides/hey-dad](https://github.com/mrbrightsides/hey-dad) and click the "Fork" button.

2. **Clone Your Fork:**
   ```bash
   git clone https://github.com/your-username/hey-dad.git
   cd hey-dad
   ```

3. **Create a Feature Branch:**
   ```bash
   git checkout -b feature/amazing-new-feature
   ```

4. **Make Your Changes:**
   - Follow the existing code style.
   - Ensure your code is type-safe (TypeScript).
   - Use Tailwind CSS for all styling.

5. **Test Your Changes:**
   - Run `npm run lint` to check for errors.
   - Run `npm run build` to ensure the project compiles.

6. **Commit and Push:**
   ```bash
   git add .
   git commit -m "feat: add amazing new feature"
   git push origin feature/amazing-new-feature
   ```

7. **Open a Pull Request:**
   Submit your PR to the `main` branch of the original repository.

## 📝 Code of Conduct

- Be respectful and kind to others.
- Focus on constructive feedback.
- Help maintain a welcoming environment for everyone.
